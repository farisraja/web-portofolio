## 🛠️ Teknologi yang Digunakan (Tech Stack)

Sistem ini tidak bergantung pada template murahan. Ini dibangun menggunakan teknologi termutakhir (Stasiun Kerja Skala Perusahaan):

### **Mesin Utama (Core):**
- **[Next.js (App Router)](https://nextjs.org/)** - *Framework* React tercepat yang memotori mesin perenderan (SSR/CSR) halaman ini tingkat dewa.
- **[React 19](https://react.dev/)** - *Library* fondasi interaktivitas UI masa kini.
- **TypeScript** - Untuk pengetikan data yang ketat dan pencegahan eror *runtime*.

### **Backend & Data (Database):**
- **[Supabase](https://supabase.com/)** - Integrasi *database/Microservices* PostgreSQL berbasis *cloud*. Mengizinkan penarikan data proyek dan sertifikat secara dinamis dari portal administrator jarak jauh.
- **GitHub REST API** - Fetch API sinkron untuk selalu menampilkan repositori kode orisinal paling segar *(fresh)* tanpa harus menyunting baris HTML.

### **Keajaiban Visual Sistem (Aesthetics & Animations):**
- **Vanilla CSS (Custom Glassmorphism)** - Merombak batasan desain datar dengan tekstur lapisan transparan efek kaca dan pendaran lampu bayangan *(Glow Orb)* yang responsif sepenuhnya.
- **[Framer Motion](https://www.framer.com/motion/)** - Menggerakkan koreografi seluruh *Scroll-Reveal* di komponen situs layaknya layar panggung teater (*Fade Up/Lift Animations*).
- **[TSParticles](https://particles.js.org/)** - Pustaka rendering *canvas* untuk membentuk jaring "Debu Partikel Galaksi" 60fps yang bereaksi mengikuti gerakan *mouse* pengunjung di pilar layar paling luar.
- **[React Type Animation](https://github.com/maxmarinich/react-type-animation)** - Mensimulasikan realitas baris kode peretas (*Typewriter Effect*) secara langsung (*Real-time*).

## ✨ Fitur-Fitur Kunci Aplikasi

1. **Light/Dark Mode (Transisi Instan):** Bosan di ruang angkasa (*Void Black*)? Pengguna bisa langsung mengubah dunia (*environment*) website menjadi tema "Kaca Es" (Frost White). Pilihan ini juga disimpan kuat di *Local Storage* sistem selamanya.
2. **Contact-To-Client:** Tidak perlu menunggu balasan email lambat. Formulir Kontak cerdas akan secara instan merangkum keluhan/pesan masuk lalu menembakkannya langsung ke *Smartphone* WhatsApp milik Faris (Fitur *Redirect URI Handler*).
3. **Scroll-Spy Adaptive Navigation:** Anda meluncur ke elemen manapun? Bilah Navigasi (*Navbar*) tidak pernah bingung; *LED Biru Tracking* akan terus mengikuti Anda sampai halaman akhir secara otomatis (pemantau rektangular otomatis berbasis *ScrollY*).
4. **Data Integrasi Sinkron:** Seluruh galeri sertifikat dan profil proyek tidak diketik pada baris kode (Hardcode), melainkan selalu dipanggil dari server (Supabase & Git Cloud) tiap detiknya.
